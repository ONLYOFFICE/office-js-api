# button

The function called when any of the plugin buttons is clicked. It defines the buttons used with the plugin and the plugin behavior when they are clicked.

| **Name** | **Data type** | **Description** |
| --------- | ------------- | ----------- |
| buttonIndex | number | Defines the button index in the &#123;@link /plugin/config#buttons buttons&#125; array of the *config.json* file. If *id == -1*, then the plugin considers that the -**Close** window cross button has been clicked or its operation has been somehow interrupted. |
