# Translations

## Scripts

### remove_unused_keys.py

Removes keys listed in `unused_translations.json` from `zh-CN.json`.

```bash
python Translations/remove_unused_keys.py
```

### add_missed_keys.py

Adds keys from `missed_translations.json` to `zh-CN.json`.

```bash
python Translations/add_missed_keys.py
```
